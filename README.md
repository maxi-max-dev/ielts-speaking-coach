# 🎙️ 雅思口语教练 · 练习版 (IELTS Speaking Coach — practice)

一个**单文件**的网页小工具：录一段英语回答 → AI 按官方四维（流利 / 词汇 / 语法）给**逐句诊断 + 估分 + Band 7-8 改写 + 下一步练习**。

> ⚠️ **重要说明**：这是 **AI 模拟**练习反馈，**不等于真实雅思成绩**；发音(Pronunciation)本版不评分。题目是 AI 按公开话题生成的「同风格练习题」，**不是真题**。
> 本工具与雅思官方（British Council / IDP / Cambridge）**无任何关联、未获其授权**，"IELTS / 雅思" 为其各自商标。仅供个人非商业练习。

## 🚀 怎么用

1. **打开网页链接**（GitHub Pages 上的 `index.html`）。
2. 点右上角 **⚙️ 设置**，填一个 AI 的 API Key（见下方「免费怎么搞」），保存。
3. 选 **Part 1 / 2 / 3** → **出题** → **开始录音**，用英语作答 → **停止**。
4. 看转写文本（可改错字）→ **获取逐句反馈 & 评分**。

> 💡 用**链接**打开就走 https，麦克风可以直接用。
> （如果是下载到本地用，请双击 `打开雅思教练.command` 走 localhost，**不要**直接双击 index.html，否则浏览器会禁麦克风。）

## 🆓 完全免费怎么搞（推荐）

打开后默认就是免费配置：服务商 = **Gemini · 免费**，转写 = **浏览器自带**。只需领一个免费 key：

1. 去 <https://aistudio.google.com/apikey>（Google 账号登录）→ Create API key
2. 复制 `AIza...` 那串 → 粘进本工具「设置」→ 保存

**不用信用卡、一分钱不花。** 免费转写请用**桌面版 Chrome / Edge**（iPhone 不支持免费转写）。

想要更强/更稳，也可在设置里切到 **OpenAI** 或 **Claude**（按量付费）。

## 🔒 隐私

- 你的 API Key **只存在你自己浏览器的 localStorage 里**，不上传任何服务器，本仓库代码里也没有任何 key。
- 录音转写与评分会发给你所选的服务（浏览器自带转写经浏览器厂商如 Google；Whisper / 评分经 OpenAI / Anthropic / Google）。
- 别把**填了 key 的本地文件**发给别人（在线链接版没这个问题，每个人填自己的 key）。

## ⚖️ 评分尺度

设置里有「标准 / 严格」（默认严格）。严格 = 像挑剔的考官，分压低、两档之间取低，适合看真实差距。
