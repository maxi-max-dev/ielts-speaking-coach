雅思口语教练 · 练习版 —— 使用说明
=====================================

【怎么打开】
双击这个文件夹里的「打开雅思教练.command」。
它会自动在本机开一个小服务器并打开浏览器（地址类似 http://localhost:8777/index.html）。
⚠️ 请保持弹出的那个黑色终端窗口开着；用完直接关掉窗口就停止了。

为什么不能直接双击 index.html？
因为浏览器规定：只有通过 https 或 localhost 打开的网页才允许用麦克风。
直接双击 index.html 走的是 file://，麦克风会被禁，录音用不了。所以用上面的启动器。
（如果你只想手动把答案打进文本框评分、不用录音，那直接双击 index.html 也行。）

【完全免费先跑（推荐，发给朋友就用这个）】
打开后默认就是免费配置：服务商 = Gemini·免费，转写 = 浏览器自带。
只需要领一个免费的 Gemini key（不用信用卡）：
1. 打开 https://aistudio.google.com/apikey （用 Google 账号登录）
2. 点 “Create API key”，复制那串 AIza 开头的 key
3. 回到本工具 → 右上「⚙️ 设置」→ 把 key 粘进去 → 「保存设置」
右上角变「✓ 已就绪 · Gemini·免费」就能用了。全程一分钱不花。
（注意：免费转写「浏览器自带」请用桌面版 Chrome / Edge；iPhone 上不支持。）

【想要更强/更稳（可选，按量付费）】
设置里把服务商切到 OpenAI 或 Claude，填对应的 key（OpenAI 以 sk- 开头、Claude 以 sk-ant- 开头）：
  OpenAI: https://platform.openai.com/api-keys ；Claude: https://console.anthropic.com
转写方式还能选「Whisper（更准）」，但 Whisper 只认 OpenAI key——
若你评分用 Claude/Gemini，要在设置里再单独填一个 OpenAI key（或转写就用「浏览器自带」）。

【评分尺度】
设置里有「标准 / 严格」。默认「严格」= 像挑剔的考官，分压得低、卡在两档之间取低的，
适合想看真实差距。想宽松点就切「标准」。

【模型填错了怎么办】
设置里「模型」一栏可改。如果出题/评分报「模型名错误」，把它清空（用默认）或换一个，
例如 Gemini 可试 gemini-2.0-flash 或 gemini-2.5-flash。

【练习流程】
① 选 Part 1 / 2 / 3，选话题（或随机）→ 点「出题」
② 点「开始录音」，对着麦克风用英语回答 → 说完点「停止」
   （Part 2 会先给 1 分钟准备，再自动开始，最多说 2 分钟）
③ 文本框会出现你说的内容，可手动改错字 → 点「获取逐句反馈 & 评分」
④ 看 AI 给的：三项估分 + 逐句诊断（哪句扣分、怎么改）+ Band 7-8 改写 + 下一步练什么
⑤ 想留着复盘就点「存入记录」，下面「练习记录」里能再看。

【重要提醒】
· 这是 AI 模拟评分，只供练习参考，不等于真实雅思分；发音这一项本版不评。
· 题目是 AI 按公开话题生成的「同风格练习题」，不是真题。
· 你的 key 只存在你这台电脑的浏览器里，不上传。但别把填了 key 的文件发给别人。
